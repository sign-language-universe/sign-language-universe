## 操作流程变化

**之前**：进入挑战 → 3-2-1-GO 倒计时 → 实时模拟评分 → 点"完成挑战" → 看结果

**现在**：进入挑战 → 点"开始"录制手语 → 比划动作 → 点"打分" → 等待评估 → 看结果（≥80分触发AR模型飞入动画）

## 关键设计决策

- 录制/打分替代实时评分：更接近真实使用场景，用户自己控制节奏
- 导航键独立于布局：方便在挑战页面快速切换词汇，无需回退
- 星球纯CSS实现：8种渐变 + 旋转光环 + 球面高光，无额外图片依赖
- 模型映射保留：已生成的5个 .glb 文件路径不变，4个待生成词汇显示emoji占位
