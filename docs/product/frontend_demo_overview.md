## 操作流程变化

**之前**：进入挑战 → 3-2-1-GO 倒计时 → 实时模拟评分 → 点"完成挑战" → 看结果

**现在**：进入挑战 → 如果词汇模板已上线则点“开始”录制手语 → 自动采集与评分 → 看结果（≥80分触发奖励展示）；如果模板未上线，则页面提示“评分模板待上线”，暂不允许录制评分。

## 关键设计决策

- 录制/打分替代实时评分：更接近真实使用场景，用户自己控制节奏
- 导航键独立于布局：方便在挑战页面快速切换词汇，无需回退
- 星球纯CSS实现：8种渐变 + 旋转光环 + 球面高光，无额外图片依赖
- 模型映射保留：已生成的 .glb 文件路径不变，无模型词汇显示 emoji 或普通奖励占位
- 评分覆盖和学习覆盖分离：学习词表全部进入挑战导航，但只有模板库覆盖词开放正式评分
